MQTT Message Communication between Pi and Server :

	From Rasberry Pi to Server:

		1)	When Item is purchased : 

			Topic         	:	item/
			Message Format	:	{"itemPurchasedX" : #current_position_x_as_string, "itemPurchasedY" : #current_position_y_as_string, cartName : #cart_name_as_string}
		
		
		2)	When Cart wants to travel from one point to other 

			Topic			:	pathOccupy/
			Message Format	:	{"fromx" : #current_x_as_string, "fromy" : #current_y_as_string, "tox" : #distination_x_as_string, "toy" : #distination_y_as_string}
		
		3)	When Cart reaches distination point

			Topic			:	pathUnoccupy/
			Message Format	:	{"fromx" : #previous_x_as_string, "fromy" : #previous_y_as_string, "tox" : #current_x_as_string, "toy" : #current_y_as_string}
		

	From Server to Rasberry Pi:
		
		1)	When server wants to send point coordinate to Rasberry Pi:
		
			Topic			:	buyItemFromServer/#cart_name_as_string
			Message Format	:	{"itemX": x_coordinate_of_item_to_buy_as_string, "itemY": y_coordinate_of_item_to_buy_as_string}
